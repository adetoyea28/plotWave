# PlotWave Function Visualizer

PlotWave is a website/API that visualizes
mathematical functions.

It uses regex patterns and the simpleeval
library to correctly parse and evaluate
functions.

# Setup Instructions
1. `git clone <repo link>`
2. `uv sync` #to install dependencies
3. `cd src`
4. `uv run main.py`

# Links

`http://localhost:8000` # API URL
